# BowlingAPI

This API includes the following endpoints:

__/start__ - start a new game

__/nextframe__ - advance one frame

__/game__ - display frames so far with interim scoring

__/oldgames__ - display all finished games in database with their IDs, final scores and individual frames


The API uses MySQL database. Database structure is available in DB_structure directory and connection is configured in __data/config.js__
